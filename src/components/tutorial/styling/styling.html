Styling the Shopping List App

Viage uses Webpack functionality to take the contents of the index.css and compile and inline it into the dest/bundle.js file. All global styling should go into the src/index.css file. Unfortunately, the more style rules you have, the bigger your DOM memory footprint and the slower the DOM will be able to render.

I recommend using inline styles for specific component functionality and reserve global styles for common styling across multiple components. However, not all CSS features are usable in inline form. Things like psuedo selectors and CSS animations just aren't possible in inline styling.

You can also use style tags in a component's HTML. The style rules will be effective across the entire document, so you have to be careful on your selector choices, but when the component goes away it's specific styling gets removed from the DOM.

Using Style Tags in a Component's HTML

To add component level styling just use normal style tags in your components HTML. Here is a simple example: 

${this.pp(this.escapeHtml(this.code1), 'html')}

In this example you can see a style tag that will be applied to shopping-list-element and some inline style in the div list container. All normal CSS rules apply and all CSS features are available as Viage does nothing to get in the way of the DOM's CSS processor.

Adding Images

Viage relies on Webpack and its plugins to manage images. You basically have 3 options when it comes to using images with Viage:

  1. Manage the images on the server by manually copying them into directory they will be served from
  2. Reference them in index.css and Webpack will automatically embed them or copy them to the dist/ folder when you do a build
  3. Import them in a Component and then reference the image as variable in a template. Webpack will copy the image or embed it depending on its size.

This tutorial will use both the CSS and importing mechanisms to demonstrate.

Importing Images

To spruce things up, we will add the Viage Logo to the ShoppingList Component and the ShoppingListAdd Component. To add it to the ShoppingListAdd Component, modify the it's HTML to match the following:

Next we need to import it into the component's TypeScript and then pass it as a parameter into the setHTML() call:
