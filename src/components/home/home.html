Viage

Simple, Minimal, Typescript Based, Modern Framework

Simple

Viage relies on common modern browser functionality and doesn't re-invent anything.

Minimal

Viage is a minimal set of base classes, best practices, and is easy to learn.

Typescript

Viage is based on Typescript which helps keep code safe, organized, and maintainable.

Modern

Viage relies on modern Common Web APIs, ES6 functionality, Typescript, and Webpack.
